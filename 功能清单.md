# 🏠 我的家 — 家庭收纳管理系统功能清单

## 基本信息

- **访问方式**：微信小程序（体验版），家人扫码即用
- **数据存储**：微信云数据库，国内稳定，多设备实时共享
- **密码保护**：进入需输入家庭密码，陌生人无法访问
- **房间数**：8 个（阳台、客厅、餐厅、主卧、次卧、厨房、卫生间、冰箱房）
- **柜子数**：58 个

---

## 一、首页 — 🔍 搜索物品

| 功能 | 说明 |
|------|------|
| 实时模糊搜索 | 输入任意关键词，即时显示匹配结果 |
| 搜索高亮 | 匹配文字黄色高亮显示 |
| 搜索结果 | 显示物品名称、所在房间、柜子、收纳盒 |
| 物品数量 | 数量大于1时显示 ×N |
| 点击查看详情 | 点击任意结果打开详情弹窗 |
| 缓存搜索结果 | 使用 IndexedDB 本地加速 |

---

## 二、新增物品 — ➕

| 功能 | 说明 |
|------|------|
| 列表选柜 | 房间 → 柜子 → 层数，级联过滤下拉 |
| 地图点选 | 切换模式，显示房间 SVG，点击柜门直接选中 |
| 物品数量 | 默认为1，支持 +/- 按钮或手动输入 |
| 已有记录提示 | 输入名称时实时显示同名物品在哪，橙色虚线圈出有同名的柜子 |
| 同柜合并 | 同柜同层加同名物品，自动叠加数量，不创建重复记录 |
| 收纳盒 | 可选，如"药箱""工具箱" |
| 备注 | 可选，补充说明 |

---

## 三、物品详情弹窗

| 操作 | 说明 |
|------|------|
| 查看位置 | 房间 → 柜子 → 层数，完整路径展示 |
| 查看数量 | 显示当前物品数量 |
| 查看时间 | 创建时间、最后修改时间 |
| 在地图中查看 | 跳转到房间完整地图，目标柜子高亮闪烁 |
| 移动物品 | 选目标房间/柜子/层 + 移动数量，支持列表选或地图点选 |
| 编辑 | 修改名称、数量、收纳盒、备注（位置只读） |
| 删除 | 数量>1时可选择删除数量，数量=1时有防误删确认 |

---

## 四、柜子地图 — 🗄️

| 功能 | 说明 |
|------|------|
| 房间选择 | 8个房间卡片，显示柜子数量 |
| SVG 矢量图 | 每个房间按实际柜子布局绘制，可点击 |
| 悬停提示 | 鼠标/手指悬停柜门，显示名称+物品数量 |
| 点击柜门 | 进入柜子详情，显示各层物品列表 |
| 物品数量 | 每个物品旁显示 ×N |
| 导航面包屑 | 房间 → 柜子，点击可返回上一级 |
| 固定设备标注 | 洗碗机、烤箱、冰箱、洗衣机用虚线框标注 |

---

## 五、设置 — ⚙️

| 功能 | 说明 |
|------|------|
| 数据统计 | 总物品数 + 各房间数量 |
| 最近更新 | 最近 5 条物品变更记录 |
| 导出 JSON | 一键下载备份文件 |
| 导入 JSON | 上传备份文件恢复（ID 去重，不覆盖已有） |
| 清空数据 | 两次确认防误操作 |
| 云同步 | Token 内置，一键检查云端 / 立即同步 |
| 版本信息 | 软件版本、数据存储方式 |

---

## 六、移动物品 — 📦

| 功能 | 说明 |
|------|------|
| 数量选择 | 全部移动或部分移动 |
| 列表选柜 | 目标房间 → 柜子 → 层数下拉 |
| 地图点选 | 显示目标房间 SVG，点击柜门选中 |
| 全量确认 | 数量全部移走时有清空原位置提示 |
| 同物合并 | 移动到已有同名物品的柜子时自动叠加数量 |

---

## 七、云同步

| 特性 | 说明 |
|------|------|
| 自动推送 | 每次增删改自动上传到云端 |
| 自动拉取 | 每次搜索前自动拉取最新数据 |
| 启动同步 | 打开应用时自动同步 |
| 双向合并 | 多设备同时修改，以最新时间为准 |
| 离线可用 | 断网时本地 IndexedDB 正常使用，联网后自动追上 |
