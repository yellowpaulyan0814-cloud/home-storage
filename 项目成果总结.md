## 五、技术架构

```
home-storage-manager/
├── index.html          ← 双击运行
├── css/
│   ├── main.css        ← 全局（Apple/Notion 极简风）
│   ├── search.css      ← 搜索页
│   ├── add.css         ← 新增页（含地图选柜）
│   ├── map.css         ← 柜子地图（含照片热区）
│   └── settings.css    ← 设置页
├── js/
│   ├── config.js       ← 房间/柜子/布局配置
│   ├── utils.js        ← 工具函数
│   ├── db.js           ← IndexedDB 封装
│   ├── router.js       ← Hash 路由
│   ├── app.js          ← 主入口
│   ├── search.js       ← 搜索页面
│   ├── add.js          ← 新增/编辑页面
│   ├── detail.js       ← 物品详情弹窗
│   ├── map.js          ← 柜子地图
│   └── settings.js     ← 设置页面
├── images/
│   ├── 客厅.png
│   ├── 餐边柜.png
│   ├── 鞋柜.png
│   ├── 主卧.png
│   ├── 次卧.png
│   ├── 阳台.png
│   ├── 厨房-水池.png
│   └── 厨房-灶台.png
└── README.md
```

### 技术栈

- 纯 HTML + CSS + JavaScript（无框架、无构建工具）
- IndexedDB 本地存储（非 LocalStorage）
- Hash 路由（SPA 单页应用）
- SVG 矢量绘制（响应式缩放）
- 完全离线，无需服务器

### 数据流

```
用户操作 → router.js 路由 → 页面模块渲染
    ↓
页面模块 → db.js → IndexedDB（读写）
    ↓
config.js（房间/柜子静态配置）
```
