# 🏠 家庭收纳管理系统（Home Storage Manager）

一个极简、快速、完全离线可用的家庭物品收纳管理网页。

**核心功能：记录物品放在哪里，需要时快速查找。**

---

## 🚀 如何运行

### 方式一：双击运行（推荐）

```
双击 index.html 即可在浏览器中打开使用
```

### 方式二：本地服务器（可选）

```bash
# 任意静态文件服务器均可
python -m http.server 8080
# 浏览器访问 http://localhost:8080
```

### 运行要求

- 现代浏览器（Chrome / Edge / Safari / Firefox 最新版）
- **完全离线**，无需网络
- **无需 Node.js、Python、PHP 等任何后端**
- 数据存储在浏览器 IndexedDB 中

## 📖 使用指南

### 搜索物品

1. 打开首页（搜索页）
2. 在搜索框中输入物品名称
3. 支持模糊搜索：输入"药"可找到"999感冒灵"、"创可贴"等
4. 点击搜索结果查看详情
5. 详情页可编辑、删除、或在地图中定位

### 新增物品

1. 点击底部"新增"标签
2. 填写物品名称（必填）
3. 选择房间 → 柜子下拉自动更新
4. 选择层数 → 根据柜子自动显示可用层
5. 可选填收纳盒、备注
6. 点击"确认添加"

### 柜子地图

1. 点击底部"地图"标签
2. 点击房间卡片进入柜子布局图
3. 鼠标悬停柜门查看物品数量
4. 点击柜门查看各层物品
5. 点击物品可查看/编辑详情

### 备份与恢复

#### 导出备份
1. 进入"设置"页面
2. 点击"导出 JSON"
3. 保存下载的 `.json` 文件到安全位置

#### 恢复备份
1. 进入"设置"页面
2. 点击"导入 JSON"
3. 选择之前导出的 `.json` 文件
4. 新数据追加，不会覆盖已有数据

## 🛠️ 自定义配置

所有房间和柜子配置集中在 `js/config.js` 文件中，修改非常方便。

### 修改柜子中文名

在 `js/config.js` 中找到 `CABINETS` 对象：

```javascript
K01: { id: 'K01', room: 'K', name: '左高柜', levels: ['L1','L2','L3','L4'], group: '电视墙' },
//                                        ^^^^^^^ 修改这里即可
```

**注意**：`id` 和 `room` 字段不要修改，它们是数据存储的关键字段。

### 新增柜子

在 `CABINETS` 对象中添加新条目：

```javascript
K12: { id: 'K12', room: 'K', name: '新柜子', levels: ['L1', 'L2'], group: '电视墙' },
```

然后在 `CABINET_LAYOUTS.K.cabinets` 数组中添加 SVG 位置：

```javascript
{ id: 'K12', x: 100, y: 200, w: 150, h: 100, rx: 6 },
```

### 新增房间

在 `ROOMS` 对象中添加：

```javascript
B: { id: 'B', name: '书房', icon: '📚', order: 6 },
```

然后在 `CABINETS` 和 `CABINET_LAYOUTS` 中添加相应柜子配置。

### 修改柜子层数

修改对应柜子的 `levels` 数组：

```javascript
// 从 4 层改为 3 层
levels: ['L1', 'L2', 'L3'],
```

1. **数据存储**：所有数据存储在浏览器 IndexedDB 中，清除浏览器数据会导致数据丢失，请定期导出备份

3. **隐私安全**：数据完全存储在本地，不会上传到任何服务器
